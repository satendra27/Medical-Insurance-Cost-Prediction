Medical Cost Insurance Prediction

Project Objective :

The goal of this project is to develop a machine learning model that predicts the medical insurance charges of an individual based on their personal and lifestyle features.
This helps both insurance providers and customers estimate costs more accurately and make better decisions.

Dataset Description:

We use a publicly available dataset (usually from Kaggle or similar), with the following features:
Feature : = Description
age : = Age of the individual
sex Gender: (male/female)
bmi: = Body Mass Index
children: = Number of dependent children
smoker: = Smoking status (yes/no)
region: = Residential region
charges: = Target variable – medical cost

Workflow & Components

  1. Data Preprocessing
    • Handling categorical variables (one-hot encoding)
    • Normalizing numerical data (optional)
    • Checking for missing values
    • Exploratory data analysis with plots
  2. Model Training
    • Algorithm tried
      • Linear Regression
      • Random Forest Regressor
      • XGBoost
    • Evaluation Metrics:
      • R 2 Score
      • Mean Absolute Error (MAE)
      • Root Mean Squared Error (RMSE)
  3. Model selection
    • The model with the best accuracy and generalization is selected (often XGBoost or Random Forest for tabular data).

Web App Implementation

Built with Flask or Streamlit + HTML/CSS for a user-friendly interface.

Pages:

  • Home – Welcome text + title
  • Prediction – Input fields for user data (age, BMI, etc.) → Predict cost
  • About Us – Team, goal, vision
  • More Details – Model info, visuals, dataset source

Technologi used

  • Python (Pandas, NumPy, Scikit-learn, XGBoost)
  • Matplotlib / Seaborn (for data visualization)
  • Flask / Streamlit (for the web interface)
  • HTML/CSS (UI layout)
  • Font Awesome (for icons)
